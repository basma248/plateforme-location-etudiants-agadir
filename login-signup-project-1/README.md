# Login Signup Project

This project provides a simple user interface for logging in, signing up, and recovering forgotten passwords. It consists of several HTML pages, CSS for styling, and JavaScript for handling user interactions.

## Project Structure

```
login-signup-project
├── src
│   ├── pages
│   │   ├── welcome.html       # Welcome page with options to log in, sign up, or recover password
│   │   ├── login.html         # Login page for user authentication
│   │   ├── signup.html        # Signup page for new user registration
│   │   └── forgot-password.html # Page for initiating password recovery
│   ├── styles
│   │   └── main.css           # CSS styles for the project
│   ├── scripts
│   │   └── main.js            # JavaScript for handling interactions
│   └── assets                 # Directory for any additional assets (images, etc.)
├── package.json                # npm configuration file
└── README.md                   # Project documentation
```

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd login-signup-project
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Open the project**:
   Open the `src/pages/welcome.html` file in a web browser to view the welcome page.

## Features

- User authentication through login and signup pages.
- Password recovery option for users who forget their passwords.
- Responsive design for better user experience.
